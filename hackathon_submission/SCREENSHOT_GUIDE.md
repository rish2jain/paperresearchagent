## üé® Screenshot Best Practices

### Technical Requirements
- **Resolution:** 1920x1080 minimum (1080p)
- **Format:** PNG or JPG (high quality)
- **File size:** Under 5MB per image
- **Aspect ratio:** 16:9 or 4:3 preferred

### Content Guidelines
1. **Show Key Features:**
   - Agent decision logs visible
   - NIM badges showing
   - Real-time updates
   - Results visible

2. **Avoid:**
   - Empty states (unless demonstrating something)
   - Error states (unless showing error handling)
   - Loading spinners without context
   - Blurry or low-resolution images

3. **Highlight:**
   - Logo and branding
   - Agent activity panel
   - Decision transparency
   - Results quality

### Editing Tips
- Add subtle annotations if needed (arrows, labels)
- Ensure text is readable
- Use consistent styling
- Crop to focus on key areas
- Remove sensitive information if any

## üöÄ Quick Screenshot Capture Guide

### Using Browser DevTools
1. Open Web UI in Chrome
2. Press F12 for DevTools
3. Use device toolbar (Ctrl+Shift+M)
4. Set to 1920x1080
5. Take screenshot (Ctrl+Shift+P ‚Üí "Capture screenshot")

### Using macOS Screenshot Tool
1. Cmd+Shift+4 for area selection
2. Cmd+Shift+3 for full screen
3. Cmd+Shift+4+Space for window capture

### Recommended Tools
- **Browser DevTools** - Best for web apps
- **Screenshot tools** - CleanSnap, Snagit, etc.
- **Screen recording** - Then extract frames
